# MMM
March Madness Modeling 

It is always a great time when March comes around and college basketball gets the spotlight.

In this project I try to apply various ideas and methods from statistics and machine learning to model March Madness.

One thing that I really like is the idea of the entire NCAA basketball landscape as a network of teams, where each team is connected to the other teams that they played this season. While it is very limited, this idea can somewhat help us gain an understanding of team strength. It could also potentially help us build sort of "team embeddings" if we apply some ideas from other fields.
